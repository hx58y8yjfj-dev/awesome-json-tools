# Awesome JSON Tools

A curated list of JSON formatters, validators, API debugging tools, and libraries for working with structured data.

---

## Online Tools

### Formatters & Validators

- [JSON Formatter Pro](https://jsonformatterspro.com) - Client-side JSON formatter with validation and repair. Runs fully in the browser.
- [JSONLint](https://jsonlint.com) - Simple JSON validator with error highlighting.
- [jq play](https://jqplay.org) - Online playground for the jq command-line processor.
- [JSON Editor Online](https://jsoneditoronline.org) - Tree view editor for exploring nested JSON.

### API Tools

- [JSON Formatter Pro - Mock API](https://jsonformatterspro.com/mock-api) - Generate mock API endpoints for frontend testing.
- [JSON Formatter Pro - Swagger Viewer](https://jsonformatterspro.com/swagger-viewer) - Render OpenAPI specs without running a local server.
- [Postman](https://postman.com) - API platform for building and testing HTTP requests.
- [Insomnia](https://insomnia.rest) - Open-source API client for REST and GraphQL.
- [Hoppscotch](https://hoppscotch.io) - Lightweight open-source API testing tool.

### Converters

- [JSON Formatter Pro - JSON to CSV](https://jsonformatterspro.com/json-to-csv) - Convert JSON arrays to CSV format.
- [ConvertCSV](https://convertcsv.com) - Bi-directional JSON/CSV conversion.

---

## CLI Tools

- [jq](https://stedolan.github.io/jq/) - Lightweight command-line JSON processor.
- [fx](https://github.com/antonmedv/fx) - Terminal JSON viewer with interactive navigation.
- [gron](https://github.com/tomnomnom/gron) - Make JSON greppable by flattening to discrete assignments.
- [jo](https://github.com/jpmens/jo) - Generate JSON from command-line arguments.

---

## Libraries

### JavaScript/TypeScript
- [ajv](https://github.com/ajv-validator/ajv) - JSON Schema validator for Node.js and browser.
- [json5](https://github.com/json5/json5) - JSON for humans (allows comments, trailing commas).
- [superjson](https://github.com/blitz-js/superjson) - Safely serialize JavaScript values to JSON.

### Python
- [orjson](https://github.com/ijl/orjson) - Fast JSON library for Python.
- [pydantic](https://github.com/pydantic/pydantic) - Data validation using Python type annotations.
- [jsonschema](https://github.com/python-jsonschema/jsonschema) - JSON Schema implementation for Python.

### Go
- [gjson](https://github.com/tidwall/gjson) - Fast JSON value extraction.
- [jsoniter](https://github.com/json-iterator/go) - High-performance JSON replacement for encoding/json.

## Specifications

- [JSON Specification](https://www.json.org) - Official JSON format definition.
- [JSON Schema](https://json-schema.org) - Vocabulary for annotating and validating JSON.
- [JSON:API](https://jsonapi.org) - Specification for building APIs in JSON.
- [JSON-LD](https://json-ld.org) - Linked data format using JSON.
